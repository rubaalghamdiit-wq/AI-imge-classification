# AI-imge-classification## Project Overview

This project was created to explore image classification using Google Teachable Machine. I trained a model using a custom dataset of jewelry images, exported it as a TensorFlow model, and tested it in Google Colab using Python.

The model can classify images into two categories:
- Ring
- Necklace

## Tools Used
- Google Teachable Machine
- Python
- TensorFlow
- Google Colab
- NumPy
- Pillow (PIL)

## Files
- `keras_model.h5` – Trained model.
- `labels.txt` – Model labels.
- `prediction.ipynb` – Notebook for making predictions.
- `prediction_example.jpg` – Example prediction from the model.
